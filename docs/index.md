## About This Site

SampleBlog is a place to share ideas, tutorials, and personal reflections.
Built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/), it's
fast, responsive, and fully accessible — with a light/dark mode toggle so you can
read comfortably at any hour.
